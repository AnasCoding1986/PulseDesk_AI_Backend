# PulseDesk AI — MERN SaaS Portfolio Project

## Project Goal
Build a premium, production-grade MERN SaaS portfolio project for Azizul Rabby (Anas), a MERN Stack Developer. The project must look like a real startup SaaS product, suitable for Upwork portfolio presentation.

## Product Concept
PulseDesk AI is an AI-powered operations command center for digital agencies, startups, and remote teams. It helps teams manage projects, clients, team performance, analytics, AI-generated insights, and client reports from one modern workspace.

## Target Users
- Digital agencies
- Startup teams
- Remote product teams
- Freelancers managing multiple clients
- SaaS operations teams

## Main Business Problems
- Teams use too many scattered tools.
- Managers lack visibility into project health.
- Agencies struggle to create client reports.
- Team performance and workload are hard to track.
- Client communication is not centralized.

## Core Features
1. Modern landing page
2. SaaS dashboard overview
3. AI insights panel
4. Project health tracking
5. Team performance analytics
6. Client management
7. Kanban-style project board
8. Report generator UI
9. Activity feed
10. Responsive mobile layout

## Important Rule
This is a portfolio project, so it must prioritize visual quality, credibility, and realistic SaaS feel.

## Avoid
- Beginner CRUD feeling
- Generic admin dashboard
- Simple todo app style
- Overcrowded UI
- Cheap colors
- Poor spacing
- Random components

## Visual Style
- Dark premium SaaS UI
- Neon blue and purple accents
- Glassmorphism used lightly
- Linear / Vercel / Stripe / Raycast inspired
- Clean spacing
- Strong typography
- Smooth animations
- High contrast
